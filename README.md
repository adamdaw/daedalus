# Daedalus Architecture Document Generator

A hashed-together repository for files used to compose architecture proposal documents.

Project uses a combination of Markdown, pandoc, mermaid, and LaTeX, as well as bibliography files generated through Zotero.

Once you have those dependencies installed and accessible, fire run.sh and it should create a project.pdf as an output.

A few other files/locations it relies on:

1. markdown/ - Individual chapter files go in here. There are a couple in the repo as examples.
2. project.bib - this is generated by Zotero, but any .bib file would do the trick. I recommend looking into markdown/.bib associations through the pandoc documentaion.
3. project.css - currently used to overwrite the default .css of the mermaid generator.
4. project.tex - used to supercede some of the standard markdown/pandoc decisions. could likely either be merged with the header fragment in the markdown folder, or that piece could be done away with altogether.
5. images/ - For Logos and other static images. Currently not working - this is my next to-do.
